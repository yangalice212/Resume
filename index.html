自傳

我叫楊允彤，成長在一個喜歡思辨、熱愛影像創作的家庭。由於父母都是紀錄片工作者，因此從小就在電影及紀錄片的薰陶下長大，除了在各式影片中獲得豐富的人生觀，也隨著父母出外拍攝取景、訪問，也讓我對事物有了更深刻的觀察與理解。

畢業於中正大學資訊管理學系，在校期間，除了 Java、C# 等程式語言外，也修了許多管理相關的課程。雖在大一上學期對課業不是很認真，導致有一門叫程式設計學科被當，但在自己重新檢討並調整步伐後，在大二下時重修了此門學科，並以接近 90 分的高分通過，同時在該學期拿到了系上前二名，得到了書卷獎的榮譽。

在畢業專題中，專案主要使用 WPF .NET Framework，我負責前端介面的設計及開發，雖然在此之前都沒接觸過前端，但在邊做邊學的過程中，看到自己設計的版面逐漸成形時，巨大的滿足感油然而生，也開啟了我對成為前端工程師的嚮往。

為了成為前端工程師，在升大四的暑假，因緣際會參加了六角學院的大學生前端體驗營，在這兩個月，從完全不會寫 HTML、CSS，到能夠熟練使用 Bootstrap 框架來撰寫具 RWD 響應式設計的網站，真的是很神奇的一段旅程，而在體驗營結束後，我也成功在 80 幾位學員中脫穎而出，成為了六角學院的實習生。

實習內容主要是擔任課程助教，負責回覆線上課程學員的問題，除了最基本的 HTML/CSS 切版技能，對於 RWD、jQuery、Bootstrap 的相關問題，也能夠簡單扼要的回覆學員。也因為在體驗營感受到在學習期間，覺得能夠有人指出自己作業的問題是一件很美好的事，所以從學習者轉換到回覆者的角色，也特別有成就感和滿足感，且在教導別人的同時，自己也是再一次的學習。

在實習期間，我也跟著六角學院學習了更進階的前端技能，除了最基本的切版能力、響應式網頁及網頁動畫效果的設計，對於 ES6 標準的 JavaScript，並透過 axios 串接 API，來製作前後台的網站也得心應手，同時也會使用 Vue.js 框架來撰寫網站，使用 Vue CLI 搭配 Vue Router 的技巧，架設具有 SPA 架構的網站。透過這些技能，我也累積了各式各樣的作品，對前端掌握度也越來越高，也在完成一次次的專案中，得到巨大的滿足感

除了六角學院的實習外，我在大學時也有過各式各樣的工讀經驗，除了擔任學校資訊處工讀生，回覆學校課程系統使用者的問題，同時整理問題並彙整成 excel 方便老師分析問題類別，並製作網站的教學文件。與此同時，因在工讀期間遇上疫情，工讀處的老師也特別指定我和另一位同學，一起處理學校遠距教學的事務，包含遠距教學軟體的功能測試，並彙整成教學文件，並且一對一為不擅長電腦的老師進行教學，這些經驗都讓我知道，工作跟機會都是要自己爭取，並展現熱情與積極的態度。透過這些工讀經驗，我也提升了解決問題的能力，並對自己更有信心。

為了將自己所學到的前端技能學以致用，立志成為前端工程師，希望能透過這個職位，在自我價值實現的同時，也能與公司一同成長，同時也希望在職場拓展自己的視野及人脈，在與不同人相處的過程中，理解他人的價值觀，並向他們學習自身沒有的經驗。
